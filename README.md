## ✨ Features

- **Single-page design** — Hero, About Me, Technical Skills, Projects, Achievements, and Blog
- **Dark theme** — Glassmorphism UI with gradient accents
- **Responsive** — Mobile-first with Tailwind CSS 3
- **Blog** — Built-in markdown-style blog with dedicated route
- **Contact** — Email, phone, WeChat, Telegram, QQ, and social links
- **Download Resume** — PDF download button in About Me section

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/simul49/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

The dev server runs at `http://localhost:5173` by default.
